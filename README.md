# 📊 CO2 Sensor Comparison Script

Analyzes CO2 sensor data from multiple sensors.

## Usage

```bash
uv run main.py
```

## Example input

```csv
DateTime,Sensor1,Sensor2,Sensor3
2026-05-22 10:00:00,400,410,420
2026-05-22 10:01:00,405,415,425
```

See [baseline_CO2_data_2026-05-22_2259.csv](data/baseline_CO2_data_2026-05-22_2259.csv)
